Gemini Prompts (Click to Copy)

Use these exact prompts in Gemini to get perfectly formatted, color-coded lyrics!

Format Style 1: Hinglish with Bracket Arrays
Main abhi jo YouTube video dekh raha hu, uske gaane ke pure lyrics Hinglish (English alphabet mein Hindi) mein nikal kar do. 
"youtubeUrl" field mein isi video ka exact link (URL) nikal kar daalna, apna koi link mat banana.
Mujhe output ek valid JSON format mein chahiye, bilkul is example ki tarah:

{
    "id": "song_shararat",
    "title": "Shararat",
    "artist": "Madhubanti Bagchi, Jasmine Sandlas",
    "youtubeUrl": "[CURRENT_YOUTUBE_VIDEO_LINK]",
    "lyrics": [
        "[0:07.0] [#ff0066] Tune parda uthaaya, haaye kya baat ho gayi",
        "[0:15.0] [#ff0066] Haaye, Ye mulakaat, vaardaat ho gayi",
        "[0:28.0] [#00ffcc] Nain milaaye humne ik dooje se yoon",
        "[0:36.0] [#00ffcc] Raat hui, Phir din, Phir se raat ho gayi"
    ]
}

Gaane ke mood ke hisaab se har line ke liye alag alag vibrant HEX colors use karna (jaise #ff4444, #00ffcc, #ffaa00).
Sirf aur sirf valid JSON block do taaki main seedha copy paste kar saku.

Mujhe output ek valid JSON format mein chahiye, bilkul is example ki tarah:

    "title": "Tu Agar Meri",
    "artist": "Arijit Singh",
    "youtubeUrl": "[CURRENT_YOUTUBE_VIDEO_LINK]",

        "(0:21 - 0:35): [#00ffff] तू अगर मेरी ये हवाएं तेरी, तू अगर मेरी सारी राहें तेरी, तू अगर मेरी मैं हूं तेरा",
        "(0:36 - 0:50): [#ff00ff] तू अगर मेरी ये उजाले तेरे, तू अगर मेरी दिल हवाले तेरे, तू अगर मेरी मैं हूं तेरा"
    ]
}

Gaane ke mood ke hisaab se har line ke liye alag alag vibrant HEX colors use karna (jaise #ff4444, #00ffcc, #ffaa00).
