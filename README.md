This project was made possible by leveraging the following incredible open-source APIs and technologies:

* **[LRCLib](https://lrclib.net/)**: For providing an exceptional, free API to auto-fetch time-synced `.lrc` lyrics.
* **[YouTube IFrame Player API](https://developers.google.com/youtube/iframe_api_reference)**: For enabling seamless background video streaming and audio playback.
